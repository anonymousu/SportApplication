Enroll for sports

This is an application where any player can register themseleves with all the details handy with them.
After registering, they can even update their details anytime.
There is an update button on player details page via which one can navigate to update page,
Else can use the update url on the header bar to navigate to update the details.
Similarly, there are register, update, search and delete links on the header bar to navigate
to respective pages.
